### 3.1 文件 / 目录的权限
目录权限
- r：可以对目录执行ls以列出目录内的所有文件
- w：可以在此目录创建或删除文件/子目录；
- x：可进入此目录，可使用ls -l查看文件的详细信息。可以理解为windows中双击就进入目录的动作

```bash
$ chmod 333 test-auth

```
