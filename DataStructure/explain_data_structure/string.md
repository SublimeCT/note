# 4-串

由零或多个字符组成的有限序列

- `string` 的逻辑结构与线性表相似
- `string` 中的元素都是字符集中的字符
- 线性表更关注的是单个元素的操作, `string` 更关注的是查找子串位置, 操作指定位置的子串


## 顺序存储结构

在一些编程语言中不直接记录 `string` 长度, 而使用 `\0` 表示字符的结束位置
