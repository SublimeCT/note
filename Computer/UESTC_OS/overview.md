# 绪论

## 用户
- Operating System Designer *操作系统设计者*: 面向操作系统 **硬件**
- Programmer *程序员*: 面向 **操作系统** 和 **开发框架**
- End User *终端用户*: 面向应用程序

## 对于 OS 的不同视角
- 从外部看
    - 计算机用户: 提供了用户的使用环境
    - 应用程序员: 提供了虚拟机 *虚拟机指虚拟化出了硬件本身没有的功能*
- 从内部看
    - OS 开发者: 资源管理视角
    - OS 开发者: 组织作业视角

## 软件系统的功能型需求
- 计算机用户使用的 `Interface` **用户接口**
    - 表现形式
        - 字符形式
        - 菜单形式
        - 图形形式
    - 使用方式
        - `on-line`
        - `off-line`
- 应用软件需要的 `System Call` **系统调用**
    - `POSIX.1`
    - `WIN32 API`
